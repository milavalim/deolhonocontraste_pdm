# deolhonocontraste_pdm
Trabalho de Programação para dispositivos móveis, uma prévia para criação do TCC. Este projeto consiste em uma ferramenta voltada para desenvolvedores e designers para facilitar a avaliação, criação e o acesso à paletas de cores apropriadas com as normas da WCAG.
