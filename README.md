# De Olho no Contraste
## deolhonocontraste_pdm
Trabalho de Programação para dispositivos móveis, uma prévia para criação do TCC. Este projeto consiste em uma ferramenta voltada para desenvolvedores e designers para facilitar a avaliação, criação e o acesso à paletas de cores apropriadas com as normas da WCAG. 
Desenvolvido por Camila Valim Dias. 

## Funcionalidades Previstas

- Cadastro e login de usuários
- Análise de contraste entre cores
- Simulação de diferentes tipos de daltonismo
- Verificação de acessibilidade baseada nas diretrizes WCAG
- Histórico de análises realizadas no perfil do usuário
- Postagem de paletas de cores em fórum
- Interface responsiva para web
- Integração com banco de dados para armazenamento das informações
- Comunicação entre frontend Flutter e backend PHP
